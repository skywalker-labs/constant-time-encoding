# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please send an e-mail to `skywalkerlknw@gmail.com`. All security vulnerabilities will be promptly addressed.

Please do not report security vulnerabilities through public GitHub issues.
