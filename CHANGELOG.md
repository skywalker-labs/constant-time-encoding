# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-07

### Added

- Initial release under Skywalker Labs.

- Rebranded vendor from `ParagonIE` to `Skywalker`.
- Updated namespace to `Skywalker\ConstantTime`.
- Updated `LICENSE.txt` copyright holder to "Skywalker Labs".
- Updated `composer.json` requirements to PHP 7.4+.

### Removed

- Removed legacy support for PHP 5.
- Removed original author attributions from source code and documentation.
